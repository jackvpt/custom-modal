Modal(props)

Modal component for displaying overlay dialogs with customizable header, content, and styles.
Parameters:
Name Type Description
props Object Component props.
Properties
Name Type Attributes Default Description
isOpen boolean Determines whether the modal is currently open.
onClose function Callback function to close the modal.
modalElements Object Elements to display inside the modal.
Properties
Name Type Attributes Description
title string Title text displayed at the top of the modal.
subtitle string Subtitle text displayed below the title.
text string <optional>
 Optional plain text message displayed in the body.
htmlElement React.ReactNode <optional>
 Optional custom HTML or JSX to render inside the modal.
closeButtonText string <optional>
 Text for the close button at the bottom of the modal (defaults to "Close").
modalOptions Object <optional>
 Optional styling and layout options.
Properties
Name Type Attributes Default Description
headerBackgroundColor string <optional>
 Background color of the modal header.
headerTextColor string <optional>
 Text color of the modal header.
backdropColor string <optional>
 Background color of the modal backdrop.
shadowed boolean <optional>
 true Whether a shadow effect is applied to the modal box.
fadeDuration number <optional>
 300 Duration of the fade-in and fade-out animations, in milliseconds.

Example
<Modal
  isOpen={true}
  onClose={handleClose}
  modalElements={{
    title: "Success",
    subtitle: "Employee created",
    text: "Employee John Doe has been added.",
    htmlElement: <p>Additional custom content here</p>,
    closeButtonText: "Dismiss"
  }}
  modalOptions={{
    headerBackgroundColor: "#4CAF50",
    headerTextColor: "#fff",
    backdropColor: "rgba(0, 0, 0, 0.5)",
    shadowed: true
  }}
  fadeDuration={500}
/>
